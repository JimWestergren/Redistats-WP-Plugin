Web stats especially made for WordPress Multisite with a large number of blogs but also works on a single blog. No additional load on your server.

== Description ==

Stats are iframed from [Redistats](https://redistats.com) - see a demo [here](https://redistats.com/demo).

= Features: =

*   No ads and no branding.
*   Global stats of all the blogs on your Multisite.
*   Stats are collected and processed in real time.
*   High privacy.
*   High performance with no additional load to you.
*   This plugin is just 3 PHP files with around 400 lines of code in total.  

== Installation ==

1.  Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
2.  Go the plugin setting and follow the steps. Requires registration on [Redistats](https://redistats.com).
